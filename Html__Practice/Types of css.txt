Types of cascadind style sheet :

1) Id selector
--------------------------------------------------------------------

<html>
    <head>
    <style>
        #some 
        {
            margin-top: 50px;
            text-align: center;
            color: red;
        }
    </style>
    </head>
    <body>
        <p id = "some"> This is effected using &ltp&gt </p>
        <p> This is not effected</p>
    </body>
</html>
